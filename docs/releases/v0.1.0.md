# Release v0.1.0

**Date:** Earlier (baseline / initial product)  
**Previous release:** —

## Summary vs previous release

First usable product line: multi-user expense tracking with a React frontend, Express API, MySQL, JWT login, dashboard charts, and monthly reporting. Deployed historically with **Vercel** (frontend) and **Railway** (backend).

## Features

- User registration and login (email + password era)
- JWT-based session for protected expense APIs
- Multi-user data isolation (`user_id` on expenses)
- Dashboard: totals, charts, recent expenses
- Add expense with categories
- Monthly report with charts
- Responsive UI; currency display in Rs

## Bug fixes

- Various production endpoint and proxy fixes for live hosting
- Multi-user login and data separation improvements

## Documentation

- Root README with setup overview

## Deployment / infrastructure

- Frontend: Vercel
- Backend: Railway
- Database: MySQL (later Aiven in production)

## Upgrade notes

- Baseline only; later releases replace hard-coded API URLs with env-based config and move auth toward username-first login.
